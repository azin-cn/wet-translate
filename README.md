## 使用示例

| command命令                  | explanation解释          |
| ---------------------------- | ------------------------ |
| wet  Hello                   | 英文->中文               |
| wet 你好                     | 中文->英文               |
| wet Hello (-D \| --dest)  CN | 翻译到中文               |
| wet hello -E2C \| --en2cn    | 指定翻译模式如英文到中文 |
